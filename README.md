# Narcizo Web Premium

Sitio web completo para publicar en GitHub Pages.

## Archivos que tenés que subir

Subí TODO el contenido de esta carpeta:

```text
narcizo-web-premium/
├── index.html
├── styles.css
├── script.js
├── menu-data.js
├── README.md
└── assets/
    ├── logo-narcizo.png
    └── video-narcizo.mp4
```

## Editar las cartas

Abrí:

```text
menu-data.js
```

Ahí podés cambiar:

- nombres de productos;
- descripciones;
- precios;
- categorías;
- productos de cada carta.

No necesitás tocar el diseño.

## Cambiar enlaces

Abrí `index.html` y reemplazá:

```text
https://wa.me/5492964000000
https://instagram.com/
https://facebook.com/
https://maps.google.com/
```

## Publicar en GitHub Pages

1. Creá un repositorio nuevo.
2. Entrá al repositorio.
3. Seleccioná `Add file`.
4. Elegí `Upload files`.
5. Arrastrá todos los archivos y la carpeta `assets`.
6. Confirmá con `Commit changes`.
7. Entrá en `Settings`.
8. Entrá en `Pages`.
9. En `Source`, elegí `Deploy from a branch`.
10. Elegí la rama `main` y la carpeta `/root`.
11. Guardá.

GitHub te mostrará la dirección pública del sitio.
